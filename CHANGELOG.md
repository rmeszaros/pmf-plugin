# Changelog

All notable changes to this plugin are documented here.

## [1.1.0] — 2026-03-07

### Added

- `pmf-validate` — assumption-level validation skill using the Extract → Map → Test workflow: surfaces 9 DVF assumptions from your riskiest PMF dimension, critiques your assumption map placement, and designs a structured experiment brief with run instructions, metrics, and success/failure signals

---

## [1.0.0] — 2026-03-07

### Added

- `pmf-hypothesis-builder` — interactive guided session for building a V1 PMF narrative across 6 dimensions with confidence scoring
- `pmf-market-research` — parallel 6-agent research workflow for identifying analogs and antilogs per dimension
- `pmf-research-synthesis` — evidence synthesis with risk scoring formula, V2 narrative update, and risk prioritization report
- `pmf-status` — read-only project state snapshot with stage detection and next-action recommendation
- Supporting assets: PMF narrative templates (structured + prose), narrative writing guide, market research synthesis template, analog/antilog criteria, risk prioritization template, synthesis framework reference
